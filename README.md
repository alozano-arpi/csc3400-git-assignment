## Description 
This repository will contain a basic calculator with arithmetic functions.

## Installation Instructions 
1. Clone the repository 
    ```sh
    git clone https://github.com/alozano-arpi/csc3400-git-assignment.git
    ```
2. Navigate to directory 
    ```sh
    cd csc3400-git-assignment
    ```
3. Run Calculator 
    ```sh
    python main.py
    ```

## Usage examples
**Run the calculator and choose from the six menu options:**
- Enter 1 for Addition
- Enter 2 for Subtraction
- Enter 3 for Multiplication
- Enter 4 for Division
- Enter 5 for Power
- Enter 6 for Square root

Example 1:
```
Choose (1-6): 1
First number: 20
Second number: 5
20 + 5 = 25
```
Example 2:
```
Choose (1-6): 4
First number: 80
Second number: 4
80 / 4 = 20
```
